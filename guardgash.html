Description

Guard and Gash its an action short game where the player is a knight fighting another knight. It is a simple combat system with delays for the inputs. So the player really has to commit on their actions. The game was a short project that was used in order to learn about game design. this is my firs game , I revisited in order to retouched a bit.

Role: Solo Dev Project
Tools: Construct 3, Adobe Fresco
